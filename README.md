# Christ House of Worship — Website Files

## Files
- `index.html` — Main church website (all sections)
- `portal.html` — Member Portal design spec mockup

## GitHub Pages Setup
1. Create a new GitHub repo called: `christ-house-of-worship`
2. Upload both files to the repo root
3. Go to Settings → Pages → Source: Deploy from branch → main → / (root)
4. Your site will be live at: `https://yourusername.github.io/christ-house-of-worship`

## Later: Custom Domain
When church is ready — buy domain like `christhouseofworship.org`
Then go to Settings → Pages → Custom domain → add it there.
